SE109: Programming In The Large

Professional Diploma of Software Engineering - Cairo University

This course introduces object-oriented programming (OOP) thinking using Java, focusing on writing large-scale, professional-quality software systems.

Course Outline & Lecture Plan

Lecture / Chapters Main Topics Key Concepts Covered
Lecture 1 (Chapters 1 & 2) Introduction to Programming & Java; Elementary Programming Programming concepts, language history, compiler vs. interpreter, Java features (OOP, portable, secure), JVM/JRE/JDK, basic program structure, errors, variables, data types, operators, console I/O, expressions, naming conventions.
Lecture 2 (Chapter 3) Selections Boolean type, relational operators, `if`, `if-else`, nested/multi-way `if`, logical operators (&&, ||, !, ^), `switch` statement, conditional operator (?:), operator precedence/associativity, common errors. Examples: Quizzes, BMI, Tax calculation, Leap Year, Lottery.
Lecture 3 (Chapter 4) Mathematical Functions, Characters, and Strings `Math` class methods (trigonometry, exponents, rounding, random), `char` type (ASCII, Unicode, casting), `Character` class, `String` class (basics, methods like `length`, `charAt`, `concat`, `toLowerCase/UpperCase`, `trim`), reading Strings/Chars, comparing strings, substrings, `indexOf`, String/Number conversions, `printf` formatting.
Lecture 4 (Chapter 5) Loops `while` loop, `do-while` loop, `for` loop, loop design strategies, sentinel values, nested loops, minimizing numerical errors, `break` and `continue` keywords. Examples: Repeated quizzes, Number guessing, GCD, Tuition calculation, Palindromes, Prime numbers.
Lecture 5 (Chapter 6) Methods Defining methods (modifiers, return type, parameters), invoking methods (arguments), method signature, formal vs. actual parameters, `void` methods, pass-by-value (primitive vs. reference), call stack, modularizing code, method overloading, ambiguous invocation, scope of local variables, method abstraction.
Lecture 6 (Chapters 7 & 8) Single-Dimensional & Multidimensional Arrays Array basics (declaration, creation, length, access), initialization, processing (input, random, printing, summing, finding min/max, shuffling, shifting), enhanced `for` loop, copying arrays (`arraycopy`), passing/returning arrays from methods, variable-length arguments (`...`), `java.util.Arrays` class (`sort`, `binarySearch`, `equals`, `fill`, `toString`), command-line arguments, 2D arrays (declaration, creation, jagged arrays, processing), multidimensional arrays.

Intended Learning Outcomes (ILOs)

On completing the course, students should be able to:

  • Carry out the analysis-design-coding cycle in writing programs using object-oriented thinking.
  • Write Java programs to solve simple and complex problems in different domains.
  • Carry out the edit-compile-run cycle in a software development environment.
