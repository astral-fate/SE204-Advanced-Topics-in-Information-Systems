Formulas and Examples

Measures of Central Tendency

Mean (Average)

Formula: x̄ = (x₁ + x₂ + ... + xn) / n

  • x̄ = Mean
  • xi = Individual data points
  • n = Total number of data points

Example (Page 31): Compute the mean for the data: 5, 6, 2, 4, 7, 8, 3, 5, 6, 6

Solution:

Sum = 5 + 6 + 2 + 4 + 7 + 8 + 3 + 5 + 6 + 6 = 52

n = 10

Mean (x̄) = 52 / 10 = 5.2

Median (Middle Value)

Formula (Odd n): Median = Value of the ((n+1)/2)th term (after sorting)

Formula (Even n): Median = Average of the (n/2)th term and the ((n/2)+1)th term (after sorting)

Example 1 (Odd n - Page 34): Find the median of: 4, 17, 77, 25, 22, 23, 92, 82, 40, 24, 14, 12, 67, 23, 29

Solution:

Sort the data: 4, 12, 14, 17, 22, 23, 23, 24, 25, 29, 40, 67, 77, 82, 92

n = 15 (odd)

Position = (15 + 1) / 2 = 8th term

Median = 24

Example 2 (Even n - Page 34): Find the median of: 3, 13, 7, 5, 21, 23, 23, 40, 23, 14, 12, 56, 23, 29

Solution:

Sort the data: 3, 5, 7, 12, 13, 14, 21, 23, 23, 23, 23, 29, 40, 56

n = 14 (even)

Positions = (14 / 2) = 7th term and (14 / 2) + 1 = 8th term

Values at positions 7 and 8 are 21 and 23.

Median = (21 + 23) / 2 = 44 / 2 = 22

Measures of Dispersion

Range

Formula: Range = Largest Observation - Smallest Observation

Example (Page 37): Find the range for the data: 3, 6, 9, 16, 27, 27, 37, 48

Solution:

Largest = 48

Smallest = 3

Range = 48 - 3 = 45

Interquartile Range (IQR)

Formula: IQR = Upper Quartile (Q3) - Lower Quartile (Q1)

  • Q1 = Median of the lower half of the sorted data
  • Q3 = Median of the upper half of the sorted data

Example 1 (Odd n - Page 39): Find the IQR for: 1, 10, 2, 3, 15, 7, 7, 8, 1, 12, 5

Solution:

Sort the data: 1, 1, 2, 3, 5, 7, 7, 8, 10, 12, 15

Overall Median (Q2) = 7

Lower Half: [1, 1, 2, 3, 5] -> Median of lower half (Q1) = 2

Upper Half: [7, 8, 10, 12, 15] -> Median of upper half (Q3) = 10

IQR = Q3 - Q1 = 10 - 2 = 8

Example 2 (Even n - Page 40): Find the IQR for: 3.3, 3.7, 2.5, 3.5, 3.0, 4.3, 3.1, 4.1, 1.9, 3.6

Solution:

Sort the data: 1.9, 2.5, 3.0, 3.1, 3.3, | 3.5, 3.6, 3.7, 4.1, 4.3

Overall Median (Q2) = (3.3 + 3.5) / 2 = 3.4

Lower Half: [1.9, 2.5, 3.0, 3.1, 3.3] -> Median of lower half (Q1) = 3.0

Upper Half: [3.5, 3.6, 3.7, 4.1, 4.3] -> Median of upper half (Q3) = 3.7

IQR = Q3 - Q1 = 3.7 - 3.0 = 0.7

Variance

Formula (Population): σ² = Σ(xi – μ)² / N

Formula (Sample): S² = Σ(xi – x̄)² / (n – 1)

  • σ² = Population Variance, S² = Sample Variance
  • xi = Individual data point
  • μ = Population Mean, x̄ = Sample Mean
  • N = Population size, n = Sample size

Example (Sample Variance - Page 47): Calculate the sample variance for: 10, 11, 15, 20, 24

Solution:

Calculate Sample Mean (x̄): (10 + 11 + 15 + 20 + 24) / 5 = 80 / 5 = 16

Calculate squared deviations from the mean:

  • (10 - 16)² = (-6)² = 36
  • (11 - 16)² = (-5)² = 25
  • (15 - 16)² = (-1)² = 1
  • (20 - 16)² = (4)² = 16
  • (24 - 16)² = (8)² = 64

Sum of squared deviations (Σ(xi – x̄)²): 36 + 25 + 1 + 16 + 64 = 142

n = 5, so n - 1 = 4

Sample Variance (S²) = 142 / 4 = 35.5

Standard Deviation

Formula (Population): σ = √[ Σ(xi – μ)² / N ] = √σ²

Formula (Sample): S = √[ Σ(xi – x̄)² / (n – 1) ] = √S²

Example (Sample Standard Deviation - Page 47): Calculate the sample standard deviation for: 10, 11, 15, 20, 24

Solution:

From the variance calculation above, Sample Variance (S²) = 35.5

Sample Standard Deviation (S) = √35.5 ≈ 5.958
